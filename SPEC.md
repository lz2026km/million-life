# 百万人生 v1.0 — 设计规格

## 1. Concept & Vision

「百万人生」是一款体彩（双色球/大乐透）历史数据分析与号码推荐工具。界面风格借鉴金融理财App的"财富感"，金色主调传达"幸运"与"机遇"。用户可选择多种分析算法，系统模拟生成推荐号码。核心体验：专业、简洁、令人信赖。

**警示**：彩票是随机游戏，历史数据仅供参考，请理性购彩。

## 4. Features & Interactions

### 4.1 彩种切换
- 双色球：红球33选6 + 蓝球16选1
- 大乐透：前区35选5 + 后区12选2
- 切换时清空推荐结果，保留历史数据

### 4.2 历史数据展示
- 近30期开奖结果
- 双色球：红球6个 + 蓝球1个（大号显示）
- 大乐透：前区5个 + 后区2个
- 支持横向滚动

### 4.3 分析算法（多选，可组合）
1. **冷热号分析** — 频率统计 + 遗漏值加权
2. **奇偶分析** — 历史奇偶比例模拟
3. **区间分析** — 号码在区间[1-11][12-22][23-33]的分布
4. **尾数分析** — 0-9尾数的热态分布
5. **机器学习模拟** — 简化版蒙特卡洛模拟（1000次随机生成，统计最优）

### 4.4 推荐号码生成
- 点击「生成推荐号码」按钮
- 根据选中的算法组合生成3注
- 球号依次弹入动画
- 每注显示算法来源标注

### 4.5 统计分析面板
- 热号TOP10（🔥标注）
- 冷号TOP10（❄️标注）
- 遗漏值排行

## 6. Technical Approach

### 架构
- 零依赖单文件 HTML（内联CSS + JS）
- 历史数据硬编码在JS中（约30期）
- 所有算法在浏览器端执行

### 数据模型
```javascript
// 双色球历史
{ period: "2026043", reds: [2,7,12,19,24,31], blue: 10 }

// 大乐透历史
{ period: "26042", front: [6,9,14,16,25], back: [3,6] }

// 推荐结果
{ reds: [3,12,18,25,29,31], blue: 13, algorithms: ["hot-cold", "odd-even"] }
```

### 算法实现
- 冷热号：频率Counter + 遗漏值权重
- 奇偶：历史奇偶比例统计
- 区间：历史区间分布统计
- 尾数：尾数出现频率分析
- 蒙特卡洛：随机生成1000注，统计各号码"中奖次数"，取TopN
